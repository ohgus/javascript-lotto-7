# javascript-lotto-precourse

## 기능 목록

### 입력

- 금액 입력
- 보너스 번호 입력.
- 당첨 번호 입력.

### 출력

- 구매한 로또 출력
  - 구매한 개수랑 같이 출력
- 당첨 통계 출력
  - 등수별 당첨 통계 및 수익률 출력

### 비즈니스

- 로또 생성
  - 1 ~ 45 사이의 무작위 숫자 6개로 이루어진 로또 생성
- 당첨 비교
- 당첨금 계산
- 수익률 계산

## 모델

### 로또

- 당첨 비교
